### Pre-checks

  docker compose version

### Getting Started

### Clone the repository
```
 git clone https://github.com/GlobalTopSystems/weather-ai
 cd weather-ai
```
### Run the Services
```
 docker compose -f weather-ai.yaml up
 docker compose -f weather-ai.yaml ps
```
